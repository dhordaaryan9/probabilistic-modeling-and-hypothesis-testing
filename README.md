# Statistics & Probability for Data Science Analysis

Project Overview
"This repository contains 15 practical tasks covering core statistical concepts applied to real-world scenarios using Python. The analysis ranges from basic descriptive statistics to complex inferential testing."

Key Concepts Covered :

  1. Descriptive Statistics: Mean, Median, Mode, Variance, and Standard Deviation.
  
  2. Probability Distributions: Binomial, Normal, and Poisson.
  
  3. Inferential Statistics: 95% Confidence Intervals and Hypothesis Testing (T-tests, Z-tests, and P-values).
  
  4. Data Visualization: Exploratory Data Analysis (EDA) using Matplotlib and Seaborn.

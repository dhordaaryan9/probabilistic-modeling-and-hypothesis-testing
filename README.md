# Probabilistic Modeling & Hypothesis Testing

This repository contains a comprehensive suite of 15 practical tasks focused on **Statistical Inference** and **Probability Theory** using Python. The project demonstrates the application of data science fundamentals to solve real-world problems, ranging from manufacturing quality control to educational performance analysis.

## 📊 Project Overview
The analysis is conducted using the standard Python Data Science stack. Each task is designed to validate a specific statistical claim using mathematical rigor and programming efficiency.

### Key Concepts Covered:
* **Descriptive Statistics:** Measures of central tendency (Mean, Median, Mode) and dispersion (Standard Deviation, Variance, Skewness, Kurtosis).
* **Probability Distributions:**
    * **Binomial:** Modeling success/failure in fixed trials.
    * **Normal (Gaussian):** Calculating probabilities and Z-scores for continuous data.
    * **Poisson:** Predicting event frequency in fixed intervals (e.g., defects per batch).
* **Inferential Statistics:**
    * **Confidence Intervals:** Calculating margins of error for both means and proportions (90% and 95% levels).
    * **Hypothesis Testing:** Implementing One-Sample and Two-Sample T-tests.
    * **P-Value Analysis:** Determining statistical significance against an alpha ($\alpha$) of 0.05.

## 🛠️ Tech Stack
* **Language:** Python
* **Libraries:** * `NumPy`: For numerical computations and vectorization.
    * `SciPy (stats)`: For advanced statistical distributions and hypothesis testing.
    * `Pandas`: For data manipulation and cleaning.
    * `Matplotlib` & `Seaborn`: For exploratory data visualization and distribution plotting.

* `main.ipynb`: The primary Jupyter Notebook containing all 15 tasks, code implementations, and interpretations.
* `requirements.txt`: List of dependencies required to run the project.
